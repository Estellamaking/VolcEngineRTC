## 目录结构说明

本目录包含 Web 版 VolcEngineRTC 的所有快速开始源代码

```
├─ Web
│  │─ ReactDemo                 // 快速开始，基于React
│  │  ├─ public                 // 图片等静态资源
│  │  ├─ src                    // 进房、通信等基础模块
│  │─ BasicDemo                 // 快速开始，基于原生JS
│  │  ├─ assets                 // 图片等静态资源
│  │  ├─ lib                    // vrtc sdk
```
